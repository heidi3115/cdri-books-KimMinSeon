# cdri-books-KimMinSeon

## 프로젝트 개요
- Kakao Book Search API 활용 도서 검색 웹사이트
- React, TypeScript, Vite
- 사용자 도서 검색, 상세 검색 및 찜하기 기능 제공

## 실행 방법 및 환경 설정
1. Node.js v20 이상 사용 (nvm use 20)
2. 의존성 설치
```bash
   npm install
```
3. 개발 서버 실행 
```bash
   npm run dev
```
4. 브라우저에서 http://localhost:5173 접속
